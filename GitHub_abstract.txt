       GitHub

1. Создали новый репозиторий (репу) имя 123.
2. У нее есть ссылка на мой аккаунт и название репозитория.
3. Программа GitHub Desktop позволяет клонировать любую репу.
4. В приложении нажимаем File → Clone repositopy → выбираем репу → нажимаем Clone
Этим действием создается на компьюторе создается папка с этой репой.
5. Кидаем файл в папку репки на компе.
6. Внизу указываем что изменилось
7. И нажимаем Commit to main
8. И нажимаем Publish branch
9. После этого у нас все это появляется на GitHub
10. Нажав на Commits можно посмотреть когда и что появилось
11. Мы наприер решили в файлике что-то написать
